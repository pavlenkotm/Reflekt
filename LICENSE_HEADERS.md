# License Headers

This document provides license header templates for different file types.

## Solidity

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.20;

/**
 * @title ContractName
 * @author Reflekt
 * @notice Brief description
 */
```

## Python

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
Module description

Author: Reflekt
License: MIT
"""
```

## TypeScript/JavaScript

```typescript
/**
 * Module description
 *
 * @author Reflekt
 * @license MIT
 */
```

## Rust

```rust
//! Module description
//!
//! Author: Reflekt
//! License: MIT
```

## Go

```go
// Package description
//
// Author: Reflekt
// License: MIT
package main
```
